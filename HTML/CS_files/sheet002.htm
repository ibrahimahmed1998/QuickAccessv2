dependency Injection باختصار مش ب assign value  جوه ال class
انما ممكن اخليها ك setter or parameter 
علشان لو احتاجت اغير البيانات محتاجش اغير جوه ال class
او جوه كل كلاس بشكل عام 
In summary, dependency injection is a specific mechanism or technique that helps achieve loose coupling in software systems. It enables components to be loosely coupled by externally providing their dependencies, resulting in more modular, flexible, and maintainable code.
